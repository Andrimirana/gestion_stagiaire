# Gestion Stagiaire

Projet web pour gérer des stagiaires développé avec **PHP**, **MySQL** et **Bootstrap**.

##  Technologies utilisées
- PHP
- MySQL
- Bootstrap 5
- HTML / CSS


##  Installation

1. Installer **XAMPP** ou tout serveur local PHP + MySQL
2. Copier le dossier `gestion_stagiaire` dans `htdocs`
3. Créer une base `gestion_stagiaire` via phpMyAdmin
4. Importer le fichier SQL (`gestion_stagiaire.sql`)
5. Modifier `connexion.php` avec les infos de la base

##  Utilisation

1. Lancer **Apache** et **MySQL** dans XAMPP
2. Ouvrir le navigateur : `http://localhost/gestion_stagiaire`
3. Le site est prêt à gérer les stagiaires
